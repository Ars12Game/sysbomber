How to install?

Linux: 
sudo apt update
sudo apt install python3 python3-pip git -y
git clone https://github.com/nordbearbotdev/Torpeda
cd Torpeda
pip3 install -r requirements.txt or python -m pip install -r requirements.txt
python3 main.py


Termux:
apt update
apt upgrade
git clone https://github.com/nordbearbotdev/Torpeda
cd Torpeda
pip3 install -r requirements.txt
python3 main.py
